# Viren Patadiya — Portfolio Website

A modern, responsive portfolio built to showcase projects, skills, and professional background.

## Tech Stack

| Layer | Technology |
|-------|------------|
| Frontend | React.js |
| Styling | Tailwind CSS, shadcn/ui |

## Sections

- **Home** — Hero landing with introduction
- **About** — Background, experience, and career overview
- **Skills** — Tech stack and tools proficiency
- **Projects** — Featured work with descriptions and links
- **Contact** — Reach out / inquiry form

## Backend & APIs

- Contact form API
- web3forms :- for send email of contact form - 250/Month mail limit

## Getting Started

Open [http://localhost:3000](http://localhost:3000) to view the site.
